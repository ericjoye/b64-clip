Why b64-clip?

Built for developers who need fast, reliable base64 encoding without leaving the terminal.

📋

Auto Clipboard

Results are automatically copied to your clipboard on macOS, Linux (xclip/wl-copy), and Windows (clip.exe).

🔄

Bidirectional

Encode plaintext to base64 or decode base64 back to plaintext — all with a single command.

📁

File & Pipe Support

Encode or decode entire files with --file, or pipe from stdin for seamless shell integration.

🔒

Zero Dependencies

Pure Python 3.11+ stdlib. No pip install needed at runtime. Nothing logged, stored, or sent anywhere.

🌐

URL-Safe Mode

Supports URL-safe base64 encoding and decoding with the --url-safe flag.

How It Works

Three steps from install to productivity.

1

Install

One command via pip, or just run the script directly. No setup, no config.

2

Encode or Decode

Pass a string, file, or pipe stdin. Supports standard and URL-safe base64.

3

Paste & Go

Result is printed and auto-copied to clipboard. Ready to use instantly.

Installation

Get started in seconds.

📦

pip install

pip install b64-clip

Recommended. Installs the CLI globally.

🐍

Run directly

python3 b64_clip.py encode "hello"

No install needed. Just run the script.

🐧

Linux clipboard

sudo apt install xclip

Required for clipboard support on Linux.
